## 走查注意

1. **创作入口**：`/compose` 只负责创建；故事创建后进大纲编辑页，角色/世界进各自详情页。  
2. **保存 vs 同步**：已发布作品「保存修改」只写本地草稿，「保存并同步市场」才更新市场展示。  
3. **大纲**：用弹窗填名称/摘要/插入位置；排序用上移、下移或「调整位置」弹窗，不再拖拽。  
4. **RAG**：不是独立功能页；世界知识库 + 世界对话自动检索。故事体验会话当前不带 `world_id`，一般不触发 RAG。  
5. **评价**：详情页仅一个「评价」区（星级 + 文字 + 列表）。  
6. **通知**：需第二账号；对自己的互动不产生通知。  
7. **PDF**：中文依赖 `storage/fonts/NotoSansSC-Regular.otf`；导出中按钮有 loading。  
8. **自动复测**：`node scripts/walkthrough-check.mjs`（依赖本地 `npm run dev` 与 `x-user-id` 调试头）。
